Personal portfolio built with Next.js, TypeScript, and Tailwind CSS.

## Description

This project is an interactive online CV with a bilingual French / English interface and dedicated sections for profile, skills, experience, projects, certifications, and contact.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion

## Features

- FR / EN bilingual site
- Smooth navigation between sections
- Light animations and transitions
- Projects and experience showcase
- Contact section

## Installation

```bash
npm install
```

## Run the project

```bash
npm run dev
```

Then open `http://localhost:3000`.

## Useful scripts

- `npm run dev` to start the development server
- `npm run build` to generate the production build
- `npm run start` to run the production app
- `npm run lint` to check the code with ESLint

## Structure

- `src/app` for pages and the global layout
- `src/components` for UI components
- `src/lib` for data and utilities
- `public` for static files
